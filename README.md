# Projektový dotahovač — dotahovac.cz

Landing page pro službu **Projektový dotahovač** od Ing. Miry Vlacha — individuální podpora seniorního projektového experta pro dotažení důležitých projektů do konce.

## O projektu

Projektový dotahovač je služba 1:1 spolupráce s projektovým manažerem s 15+ lety praxe. Klient získá plánovací konzultaci, rozpad projektu na zvládnutelné úkoly a pravidelné týdenní check-iny až do dokončení projektu.

Web prezentuje:
- Jak spolupráce funguje (4 kroky)
- Ceník (individuální spolupráce 10 000 Kč/měsíc, DIY balíček 500 Kč)
- Reference klientů
- O autorovi
- FAQ
- Kontaktní CTA

## Technologie

- Statický single-page HTML
- CSS custom properties, grid layout,Responzivní design
- Google Fonts (Outfit, Playfair Display)
- Vanilla JavaScript (scroll efekty, FAQ accordion, reveal animace)

## Spuštění

Stačí otevřít `index.html` v prohlížeči — žádný build ani server není potřeba.
